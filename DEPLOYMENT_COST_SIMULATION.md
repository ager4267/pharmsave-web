# 웹 배포 비용 시뮬레이션

이 문서는 **팜세이브(PharmSave)** 프로젝트를 웹에 배포할 경우 예상되는 월간 유지 비용을 시뮬레이션합니다.

## 프로젝트 개요

- **기술 스택**: Next.js 14, TypeScript, Supabase (PostgreSQL, Auth, Storage)
- **주요 기능**: 
  - 사용자 인증 및 관리
  - 파일 업로드 (Excel, 문서)
  - 데이터베이스 (PostgreSQL)
  - 이메일 알림
  - 실시간 데이터 처리
